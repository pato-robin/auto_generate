Chapter 2: Automatic Generation for Different Aspect Ratios and Orientation

Overview

This chapter demonstrates how the system can take a single product image and automatically generate background variations at different aspect ratios and orientations. This is essential for producing assets tailored to various platforms and display formats from one source image.

Input

Out1.png — The input image used as the base for generating different aspect ratio outputs.

Outputs

16:9

Out2_1.png — 16:9 landscape format.

21:9

Out2_2.png — 21:9 ultra-wide format.

32:9

Out2_3.png — 32:9 super ultra-wide format.

9:21 (Vertical)

Out2_4.png — 9:21 vertical / portrait format.

How It Works

  1. Input — Provide a product image with a generated background (e.g., Out1.png).
  2. Aspect Ratio Selection — Choose the target aspect ratios: 16:9, 21:9, 32:9, 9:21, etc.
  3. Generation — The system extends or adapts the background to fill the new canvas while keeping the product properly framed.
  4. Output — One image per aspect ratio is produced, ready for use across different platforms and displays.
